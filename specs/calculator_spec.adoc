==  Addition
<!-- Addition de deux nombres positifs simples -->
[source,gherkin]
----
Feature: Addition
  Scenario: Addition de 5 et 3
    Given un objet Calculator
    When j'appelle add avec 5 et 3
    Then le résultat doit être 8

  Scenario: Addition de -5 et 3
    Given un objet Calculator
    When j'appelle add avec -5 et 3
    Then le résultat doit être -2

  Scenario: Addition de 2.5 et 3.1
    Given un objet Calculator
    When j'appelle add avec 2.5 et 3.1
    Then le résultat doit être 5.6
----
